Web Application Style Guide

This style guide provides a reference for the visual style of the web application, including colors, typography, and other design elements.

Color Palette

The color palette for the application has been designed to convey professionalism, trustworthiness, clarity, user-friendliness, consistency, responsiveness, security, and strong branding. The colors should be used consistently across the application to ensure a cohesive visual experience.

Primary Blue

#2E5A9A

Use for primary elements like headers, buttons, and main navigation.

Secondary Blue

#7896C1

Use for secondary elements like hover states, tooltips, and highlights.

Accent Grey

#A1AAB3

Use for accents, borders, and subtle backgrounds.

Light Background

#F3F4F6

Provides a clean background for content, while being easier on the eyes than pure white.

Dark Grey Text

#2D2F32

Use for body text, ensuring readability and reducing eye strain compared to pure black.

Medium Grey Text

#74787C

Use for secondary text elements, such as labels, captions, or less important information.

Alert Green

#3C9465

Use for success messages or positive indicators.

Alert Red

#C0392B

Use for error messages or negative indicators.
