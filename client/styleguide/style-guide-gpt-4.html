Web Application Style Guide

This style guide provides a reference for the visual style of the web application, including colors, typography, and other design elements.

Layout and Whitespace

A well-organized layout with appropriate whitespace ensures easy readability and a clean appearance. The following guidelines and CSS variables can be used to maintain a consistent layout and whitespace throughout the application.

Container

The container class can be used to center and set a maximum width for the content:

.container {
max-width: var(--container-max-width);
margin-left: auto;
margin-right: auto;
padding-left: var(--container-padding);
padding-right: var(--container-padding);
}

Margins and Padding

Use the following CSS variables to apply consistent margins and padding:

element {
margin: var(--margin-default);
padding: var(--padding-default);
}

Accessibility

Ensuring that the application is accessible to all users, including those with disabilities, is a priority. Follow these guidelines to maintain accessibility in the application:

  • Use high-contrast color schemes for easy readability.
  • Provide alt text for all images using the alt attribute.
  • Ensure keyboard navigation for all interactive elements.
  • Use semantic HTML elements to convey structure and meaning.
  • Employ ARIA attributes to improve the accessibility of custom interactive components.
  • Ensure all form controls have associated <label> elements.
  • Make the application responsive to work well on different devices and screen sizes.
  • Regularly test with popular screen readers like JAWS, NVDA, or VoiceOver.
